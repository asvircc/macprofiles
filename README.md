# macprofiles
mac profile, .zshrc .vimrc  etc.
---
##.zshrc
1. Change default shell
```shell
chsh -s $(which zsh)
```

2. Modify `.zshrc`

##.vimrc
1. use [vundle](https://github.com/gmarik/Vundle.vim)
```shell
git clone https://github.com/gmarik/Vundle.vim.git ~/.vim/bundle/Vundle.vim
```

2. Set up Vundle
